# HexToDecimal.app

Static SEO website for [hextodecimal.app](https://hextodecimal.app), focused on practical number-base converters and calculators.

## Local Development

```bash
npm install
npm run build
npm run dev
```

The site is generated into `dist/` and is intended for Cloudflare Pages.

## Pages

The site includes dedicated English pages for hex, decimal, binary, ASCII conversion, and calculator search intents. Each converter page is statically rendered for SEO and uses browser-side JavaScript for instant calculations.
